# rapid-crisis-response
The Rapid Crisis Response System is an AI-powered solution for real-time emergency detection and response. It uses CCTV, IoT sensors, and a panic button to identify fire or distress and instantly alert a central dashboard, enabling quick, coordinated action and improved safety.


# 🚨 Rapid Crisis Response System (AI-Based)

## 📌 Overview
An AI-powered system that detects emergencies (fire, sound, panic alerts) and sends real-time notifications through a unified dashboard.

---

## 🎯 Features
- 🔥 Fire/Smoke Detection (OpenCV)
- 🔊 Sound Detection
- 📱 Panic Button
- 📊 Live Dashboard
- 🔔 Notifications
- 🗺️ Map Integration
- ☁️ Firebase Real-Time Database

---

## 🏗️ Tech Stack
- Python (OpenCV, Sound Detection)
- Node.js (Backend API)
- Firebase (Database)
- HTML, JavaScript (Frontend)
